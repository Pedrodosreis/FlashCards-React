# Flash Cards Project

The application is designed to help people study using cards. The front of the card has a question and the back part of the answer. That way, students or anyone who needs to develop any skills can use the app to study.

This project was tested using Android.

Start Project

* To run the Project 
    - `git clone -urlRepository-` 
    - `cd flashCards`
    - `npm install`
    - `npm start`
